== Description ==

Podcast Episodes es un plugin de WordPress que te permite crear y gestionar episodios de podcasts de manera fácil y eficiente. Con este plugin, puedes crear un Custom Post Type llamado 'podcast_episodes', una taxonomía personalizada llamada 'Podcast' y agregar enlaces a diferentes plataformas de reproducción como YouTube, Spotify e iTunes Podcast.

Características principales:

* Custom Post Type 'podcast_episodes' para manejar episodios de podcast.
* Taxonomía personalizada 'Podcast' para organizar y categorizar los episodios.
* Campos personalizados para almacenar las URLs de YouTube, Spotify e iTunes Podcast.
* Botones y enlaces para ver en YouTube, escuchar en Spotify y escuchar en iTunes Podcast en la lista de la taxonomía y debajo del título del episodio.
* Compatible con el editor Gutenberg.

== Installation ==

1. Sube la carpeta `podcast-episodes` al directorio `/wp-content/plugins/` de tu sitio web.
2. Activa el plugin a través del menú 'Plugins' en el panel de administración de WordPress.
3. Empieza a crear y gestionar episodios de podcasts utilizando el nuevo Custom Post Type 'podcast_episodes' y la taxonomía 'Podcast' en el panel de administración.
